# leetcode-sql-challenges
Folder ini berisi 10 soal SQL pilihan dari LeetCode untuk menguji kemampuan analisis dan menemukan solusi pemecahan masalah soal.
Adapun sub kategori soal leetcode yang termasuk dalam folder ini adalah sebagai berikut:
1. `SELECT `
2. `BASIC JOINS`
3. `BASIC AGGREGATE FUNCTION`
4. `SORTING and GROUPING`
5. `ADVANCED SELECT and JOINS`
6. `SUBQUERIES`
